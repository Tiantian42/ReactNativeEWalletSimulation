# COINNECT E-WALLET SIMULATION
   
DESCRIPTION
-----------
This is a team project e-wallet application simulation built 
using the React Native framework. The application provides 
a comprehensive interface for managing digital assets, 
featuring advanced tracking tools for the modern crypto user.

KEY FEATURES
------------
* Real-Time Crypto Tracker: Live price updates for various 
  cryptocurrencies.
* Gas Fee Tracker: Monitor network transaction costs in 
  real-time to optimize transfers.
* E-Wallet Simulation: Interactive UI for viewing balances 
  and transaction simulations.
* Cross-Platform: Designed for both iOS and Android.

TECH STACK
----------
* Framework: React Native
* Language: JavaScript
* Data Sourcing: External Crypto & Gas APIs
